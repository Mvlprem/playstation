## Playstation 5 Clone

This project focuses on to create a responsive Playstation 5 website format as close as possible.

### Demo : [Playstation](https://prem-playstation.netlify.app/)

### Getting started

```
git clone https://github.com/Mvlprem/playstation.git
cd playstation
npm install
npm run dev
```
